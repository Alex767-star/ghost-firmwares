Участие в проекте Ghost Firmwares
Как сообщить об ошибке

    Проверьте существующие Issues

    Создайте новый Issue с шаблоном баг-репорта

    Укажите: устройство, версию прошивки, шаги для воспроизведения

Как предложить улучшение

    Откройте Issue с меткой enhancement

    Опишите функцию и зачем она нужна

    Если есть пример кода — приложите

Pull Requests

    Форкните репозиторий

    Создайте ветку: feature/имя-фичи

    Пишите код в стиле проекта

    Проверьте сборку: make all

    Откройте PR в ветку main
